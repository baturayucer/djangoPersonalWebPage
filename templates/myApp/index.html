Spring Boot Online Reservation App

Keywords: SpringBoot, Spring Security, Docker, Travis-ci

There are two calendar views as weekly and monthly.
Users are able to book on a daily and hourly basis.
Reservations have been kept in a stable database and the connection between the calendar view using rest services and synchronization have been established.
